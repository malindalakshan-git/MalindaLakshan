# Malinda Lakshan — Portfolio

Personal portfolio website built with vanilla HTML, CSS, and JavaScript. Hosted on GitHub Pages.

## Pages

- **Home** (`index.html`) — Hero, skills matrix, featured projects
- **About** (`about.html`) — Bio, skills grid, interests
- **Research** (`research.html`) — Current research in Buddhist philosophy & logic
- **Projects** (`projects.html`) — Technical projects with screenshot gallery
- **Contact** (`contact.html`) — Contact form via Web3Forms, social links
- **404** (`404.html`) — Custom error page

## Features

- Dark/light theme toggle (persisted to localStorage)
- Neural network particle background canvas
- Typewriter animation on hero
- Scroll-triggered reveal animations
- 3D card tilt & magnetic button effects
- Page transition overlay
- Project detail modal with image gallery
- Skills detail modal
- Keyboard shortcuts (`T` theme, `?` shortcuts)
- Toast notification system
- Responsive design (mobile-first)

## Tech Stack

- HTML5
- CSS3 (custom properties, animations, glassmorphism)
- Vanilla JavaScript (ES6+)
- FontAwesome 6.4 (icons)
- Google Fonts (Inter, Outfit, Space Grotesk)
- Web3Forms (contact form backend)

## Deployment

Deployed via GitHub Pages — push to `main` branch.

## Security

- CSP enforced via meta tag
- Clickjacking protection (`frame-ancestors 'none'`)
- HTTPS enforced by GitHub Pages
- No secrets in git history
